TLS 1.3 middleboxes test

This page performs some tests to check for middlebox interference with TLS 1.3. For that it requires Adobe Flash and TCP port 843 to be open. If this is not the case, all tests will fail with N/A.

Enable additional MITM detection, this requires the server to record test results that could be sent to Cloudflare for anlytical purposes. Learn More

The default test information that we collect:
- client IP address
- the result (pass/fail) of a connection (to know which networks/ISPs are affected by the problem)
- the synthetic TCP payloads and session keys (these are used to investigate the cause for failing connections)

The optional of information that we collect:
- user agent (web browser)
- flash version
